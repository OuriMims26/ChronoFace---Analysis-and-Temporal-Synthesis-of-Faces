# ChronoFace---Analysis-and-Temporal-Synthesis-of-Faces
ChronoFace : Projet d'IA alliant prédiction d'âge (Swin Transformer, MAE 6 ans) et vieillissement facial réaliste (CycleGAN entraîné from scratch). Développée en Python/PyTorch et déployée sur Streamlit, cette solution transforme les visages tout en préservant l'identité de l'utilisateur.
